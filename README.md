## Utilisation du template

Ce template peut être utilisé aussi bien avec **PLMLaTeX** et **Overleaf**, sans aucun prérequis, que avec des éditeurs LaTeX conventionnels.

---

## Utilisation rapide : Overleaf & PLMLaTeX en ligne

1. Ouvrir le dépôt GitHub : [Template LaTeX du Futur](https://github.com/aallonas/Template_LaTeX_du_Futur)
2. Cliquer sur le bouton **Code**
3. Cliquer sur **Download ZIP**
4. Créer un **nouveau projet** sur Overleaf ou PLMLaTeX
5. Choisir l’option **Importer un projet**
6. Importer le fichier ZIP téléchargé depuis GitHub

Cette archive contient l’ensemble des fichiers nécessaires au fonctionnement du template.  
Une fois importé, `main.tex` est le fichier principal à compiler.
